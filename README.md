## 4. Вариант задания
**Текстовое описание:** Распознавание лексем языка программирования Python (условный оператор `if-else`).

**Перечень допустимых лексем:**
1.  **Целое без знака:** последовательность цифр (0-9).
2.  **Идентификатор:** последовательность букв латинского алфавита и цифр, начинающаяся с буквы.
3.  **Ключевые слова:** `if`, `else`.
4.  **Операторы:**
    - Оператор присваивания: `=`
    - Операторы сравнения: `>`, `<`, `>=`, `<=`, `==`, `!=`
5.  **Разделители:**
    - Пробел ( ` ` )
    - Двоеточие ( `:` )
    - Символ новой строки ( `\n` ) — учитывается для многострочности.
    - Отступы (пробелы/табуляция в начале строки учитываются как часть структуры, но могут обрабатываться как разделители).
6.  **Конец оператора:** `;` (точка с запятой).

**Примеры корректных входных строк:**
```python
if a > b:
  max = a
else:
  max = b;
```
